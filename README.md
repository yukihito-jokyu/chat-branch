# chat-branch

## プロジェクト概要

ChatGPTやGeminiなどのチャット機能で生成された文字に対して深掘りをする場合、既存のチャットの下で聞くか、新規チャットを開いてそこで聞くかの方法があります。
それぞれには問題があり、既存のチャットの場合はその深掘りが不要・もしくはノイズになった場合、その後に生成される文章の質が下がってしまうことです。
新規チャットの場合は、その生成された文章を既存のチャットでも活かせなくなることです。
そうした問題を解決するためにチャットの階層構造化を行うことができるwebアプリケーションを作成したいです。

### アイデア：チャットの階層構造化

本プロジェクトでは、
「会話の一部に対して独立した深掘りチャットを階層的に生やす」
という新しいチャット体験を提供する Web アプリケーションを実装する。

### コアとなるコンセプト

元チャットの特定の単語・文を選択すると、その内容を起点に
1 つ下の階層でサブチャットを生成できる。

サブチャットは独立したコンテキストを持つため、
本筋に影響を与えず自由に深掘りできる。

深掘り完了後、ユーザーは

- 上の階層に 反映する要点を選択して戻す
- 不要であれば 切り捨てて破棄する

といった操作ができる。

これにより、
「深掘りの自由度」 と 「本筋の品質の維持」 を同時に実現する。
